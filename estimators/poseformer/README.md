# PoseFormer

## Clone their repo
git clone https://github.com/zczcwh/PoseFormer.git poseformer

## Follow their README
Download files from Google Drive

## Create and activate environment
conda activate /shares/iict-sp2.ebling.cl.uzh/cobrie/pose_estimation_comparison/venvs/poseformer

## Put the video you want in poseformer_demo/demo/video

## Run the demo 
From the base of the poseformer directory, run:
python demo/vis_poseformer.py --video YOUR_VIDEO_NAME.mp4

## Access the results
The resulting .mp4 visualization will be in the directory output/YOUR_VIDEO_NAME/
